# Data notice

This project derives indexes from:

- Oqaasileriffik / Greenlandic Language Secretariat, via the public Katersat
  lexicon export: <https://jandahl.github.io/Oqaasileriffik-katersat/lexicon.json>
- Katersat semantic classes: <https://jandahl.github.io/Oqaasileriffik-katersat/semantic_classes.json>

this project is also made available under CC BY-SA 4.0:
<https://creativecommons.org/licenses/by-sa/4.0/>.
